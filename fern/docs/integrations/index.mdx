# Integrations

Promptless integrates with your existing tools to automate documentation workflows.

Integrations serve three purposes: 
1. **Triggers** - Generate events that notify Promptless (automatically or manually)
2. **Context** - Let Promptless search for relevant information during research
3. **Publishing** - Push documentation updates to your CMS

Some integrations serve multiple purposes. Beta integrations may not be listed here. Contact help@gopromptless.ai for the latest list.

## Available Integrations

- [Bitbucket](/docs/integrations/bitbucket-integration): Trigger on pull requests. Publish to Bitbucket repos.
- [GitHub](/docs/integrations/github-integration): Trigger on pull requests. Research code functionality. Publish to GitHub repos.
- [GitHub Enterprise](/docs/integrations/github-enterprise-integration): Custom GitHub App for Enterprise Server and Enterprise Cloud.
- [Jira](/docs/integrations/jira-integration): Search Jira issues and project data using JQL.
- [Linear](/docs/integrations/linear-integration): Search Linear issues and projects within teams.
- [Microsoft Teams](/docs/integrations/microsoft-teams-integration): Trigger on mentions and message actions.
- [Slack](/docs/integrations/slack-integration): Trigger on DMs or message actions.
- [Zendesk](/docs/integrations/zendesk-integration): Trigger on resolved tickets. Publish to Zendesk help center.

### Beta Integrations

- [Intercom](/docs/integrations/intercom-integration-beta): Publish to Intercom help center. Contact help@gopromptless.ai for access.
- [Webflow](/docs/integrations/webflow-integration-beta): Publish to Webflow CMS, including blog posts and collection items. Contact help@gopromptless.ai for access.

## Security & Authentication

All Promptless integrations follow security best practices:
- OAuth 2.0 authentication where applicable
- Encrypted data transmission (TLS 1.2+)
- Encrypted data storage (AES-256)
- Minimal data retention

## Common Features

All integrations include:
- Granular permission controls
- Audit logging
- Configurable notifications
- Data privacy controls

For security details, see [Security and Privacy](/docs/security-and-privacy).
