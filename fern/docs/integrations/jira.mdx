# Jira Integration

**Used for: Context**

Promptless integrates with both Jira Cloud and Jira Data Center instances.

## Integration Types

Promptless supports two types of Jira integrations depending on your Jira deployment:

<CardGroup cols={2}>
  <Card title="Jira Cloud" icon="fa-solid fa-cloud">
    For Atlassian-hosted Jira instances. Uses OAuth 2.0 authentication through Atlassian's secure connection process.
  </Card>
  <Card title="Jira Data Center (Beta)" icon="fa-solid fa-server">
    For self-hosted Jira Data Center installations. Uses username and password authentication with your organization's instance.
  </Card>
</CardGroup>

<Info>
Jira Data Center is in Beta. Please contact the Promptless team at [help@gopromptless.ai](mailto:help@gopromptless.ai) to connect your Jira Data Center instance to Promptless.
</Info>

## JQL Search Capabilities

Promptless leverages Jira's JQL (Jira Query Language) to find relevant context:

- **Automatic Issue Retrieval**: When a GitHub PR or other trigger mentions a Jira ticket (like "PROJ-123"), Promptless automatically retrieves that issue.

- **Proactive JQL Searches**: Promptless can search Jira using JQL queries to find related issues, epics, and project data.

<Note>
Promptless queries Jira data in real-time. It does not store your Jira data. All searches happen on-demand.
</Note>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Connection Issues">
    If you're having trouble connecting:
    - Ensure you have necessary permissions in your Jira instance
    - Check that your Jira instance allows OAuth 2.0 applications
    - Contact your Jira administrator for permission errors
  </Accordion>

  <Accordion title="Search Results">
    If Promptless isn't finding relevant issues:
    - Verify projects are configured correctly in your Promptless project settings
    - Check that issues exist in the configured projects
    - Ensure issues are accessible with your current permissions
  </Accordion>

  <Accordion title="Re-authentication">
    To re-authenticate with Jira:
    - Visit the [integrations page](https://app.gopromptless.ai/integrations)
    - Delete and reconnect your Jira integration
  </Accordion>
</AccordionGroup>
